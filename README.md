# DotNet-GUI-Projects

Welcome to my C# .NET GUI showcase repository! 🚀

This repository is dedicated solely to building and exploring user interface (UI) designs and desktop application layouts using the .NET Framework. 

### 📌 Features:
* **Focus on UI/UX:** Clean, responsive, and functional desktop designs.
* **Pure Code:** All projects run independently with pure backend logic.
* **No Database:** No external dependencies, heavy configurations, or database setup required—just open and run.

Feel free to explore the different app concepts and layouts!
